PanPG demo 2010-04-05

This is a very simple demo that demonstrates the PanPG API. We load a PEG from a file, generate a parser, use that parser to parse some input, and finally display the resulting parse or error. You can edit the arithmetic expression below and see the different parse trees generated. You can also edit the grammar, and the parser will be regenerated and the input parsed again.

The parser generator also has Unicode support, including characters outside the BMP, though this demo doesn't use them.

This is the new codegen, you can see the previous version, or read the project roadmap. There's also an an ECMAScript parser demo.
